# Newsletter-Finder
Расширение которое ищет рассылки

------

Чтобы его установить:
- нужно скачать CRX
- Включить режим разработчика в Chrome или подобном браузере
- Перекинуть CRX файл в браузер во вкладку "Расширения"

------

Что делает расширение: сканирует страницы браузера, для поиска рассылок, чтобы в дальнейшем подписаться на них.
Зачем оно было создано? Для фарминга аккаунтов, для заполенения почтового ящика
